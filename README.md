Descripción del proyecto
Helenic Works es una aplicación web para un gimnasio. Este trabajo práctico agrega un backend con Node.js y Express que expone una API REST, reemplazando los datos estáticos del TP1 por datos dinámicos obtenidos desde el servidor.

- Frontend: https://joacopelizza.github.io/TP3-Helenic-Works
- Backend (API): https://tp3-helenic-works.onrender.com

Metodología de trabajo con Git y GitHub
Se trabajó con ramas por integrante. Cada integrante creó su propia rama, realizó sus commits ahí y luego hizo un pull request para mergear a `main`.
